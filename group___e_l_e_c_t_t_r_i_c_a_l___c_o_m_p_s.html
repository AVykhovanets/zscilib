int zsl_phy_ecmp_capac_ad ( zsl_real_t  a,
zsl_real_t  d,
zsl_real_t *  c 
)

Calculates capacitance in farads based on the permitivity of freespace multiplied by area (a) and divided by distance (d). i.e.: C = E (A / D) where E = 8.85 x 10^-12 farad per meter (F/m).

This function can be used to calculate the value of a parallel plate capacitor, consisting of two metallic plates of area a, separated by a vacuum with a gap of d.

Parameters
aArea in square meters.
dDistance in meters.
cPointer to the output capacitance in farad. Will be set to NAN if d = 0.
Returns
0 if everything executed properly, error code on failure.
